Code spotify

Vous pouvez obtenir un code Spotify (l'espèce d'onde sonore qui apparaît sous le visuel du contenu) à partir de n'importe quel profil, titre, album ou playlist sur l'appli. Cliquez sur les trois points, puis sur la pochette de l'album contenant le code et sur Enregistrer dans la galerie.

Créer un Code Spotify :

Rendez-vous sur le site Spotify Codes, collez le lien dans la zone de texte et cliquez sur « Obtenir le Code Spotify ». Personnalisez les couleurs, la taille et le format de l'image (SVG, PNG, JPEG), puis téléchargez le code.
